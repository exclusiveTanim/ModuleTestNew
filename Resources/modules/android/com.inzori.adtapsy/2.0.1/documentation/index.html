AdTapsy Module

Description

This module allows to integrate AdTapsy sdk into your Titanium Project. If you want to know about AdTapsy check their website: http://www.adtapsy.com.

You have to get your appId key by registering with AdTapsy.

NOTE: We currently support Interstitial (fullscreen) ads only. No banners are supported.

Accessing the AdTapsy Module

To access this module from JavaScript, you would do the following:

var adtapsy = require('com.inzori.adtapsy');

The adtapsy variable is a reference to the Module object.

Check app.js for complete example.

METHODS

This module exposes these methods:

init

Initializes the module.

Parameters:
    - appId: get yours from AdTapsy website by registering an app.

showInterstitial

Shows an interstitial ad.

EVENTS

adtapsy:init

Fired by the init method.

adtapsy.addEventListener('adtapsy:init', function(e){
    Ti.API.info(e);
});

IMPORTANT 1 !!

You have to create an account with each ad provider you want to use. (AdMob, Chartboost, etc). Once you have the accounts created, head to adtapsy.com, create an account if you don't have one already, create an application and select which networks you want to integrate.
Enter the required data then. For some networks it's just an API key, for other networks you may have to enter your credentials or link to your AdSense account.

IMPORTANT 2 !!

If you are willing to integrate Vungle into your application, you HAVE to copy the file "__vungle.db" provided with this module into your Resources folder. It will break with a "database not found" error otherwise.

IMPORTANT 3 !!

I think you already read this above but just to make it clear: We currently support Interstitial (fullscreen) ads ONLY.
