emailprovider Module

Description

Get the primary logged in email of the android device.

Accessing the emailprovider Module

To access this module from JavaScript, you would do the following:

var emailFetcher = require('com.emailprovider');

The emailFetcher variable is a reference to the Module object.

Usage

var emailFetcher = require('com.emailprovider');

//Returns the primary email of the device or Return null if no email setup//

var primaryEmail = emailFetcher.getPrimaryEmail();

Ti.API.info('Device primary logged in email id :' + primaryEmail);
